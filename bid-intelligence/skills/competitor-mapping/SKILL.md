# 경쟁사 매핑 분석

입찰에서 예상되는 경쟁사를 체계적으로 분석하여 수주 전략 수립을 지원한다.

## 왜 경쟁사 분석이 필요한가

입찰은 상대 평가다. 제안서가 아무리 좋아도 경쟁사보다 나아야 이긴다. 경쟁사의 강점을 알아야 차별화 포인트를 설계할 수 있고, 약점을 알아야 우리의 상대적 우위를 부각할 수 있다.

## 컨텍스트 활용

이 스킬을 실행하기 전에, 프로젝트의 `_context/` 폴더가 있는지 확인한다.

**폴더가 있으면:**
1. context-loader로 회사 프로필 구축
2. `_context/proposals/`의 과거 제안서에서 **경쟁사 이력**을 참조 — 이전에 어떤 업체와 경쟁했는지, 결과는 어떠했는지
3. 회사 역량 데이터를 경쟁 매트릭스에 자동 반영
4. 정량 비교 (실적 건수, 매출, 인력 수)를 데이터 기반으로 수행

**폴더가 없으면:**
- 사용자에게 회사 역량 정보를 질문
- 경쟁사는 사용자 입력 + 웹 검색으로 파악

## 분석 절차

### 1단계: 경쟁사 후보 식별

다음 소스에서 예상 경쟁사를 식별한다:

- **과거 유사 사업 수주 이력**: 해당 분야에서 최근 3~5년간 수주한 업체
- **_context/ 과거 제안서**: 이전 입찰에서 만난 경쟁사 이력 (있는 경우)
- **사업설명회 참석사**: 공개된 경우 직접 확인
- **업계 네트워크**: 해당 분야의 주요 플레이어
- **웹 검색**: 최근 낙찰 공고, 수주 뉴스

사용자에게 알고 있는 경쟁사 정보를 확인한다.

### 2단계: 경쟁사별 역량 프로파일

각 경쟁사에 대해 다음 6개 영역을 분석한다:

**A. 유사 실적**
- 해당 분야 실적 유무 및 건수
- 해당 발주처 실적
- 최근 수주 트렌드 (상승/하락)

**B. 핵심 인력**
- 확보하고 있는 핵심 전문가 풀
- PM급 인력의 해당 분야 경험

**C. 네트워크**
- 관련 기관과의 기존 관계
- 현지/대상 지역 파트너 유무

**D. 기술 역량**
- 핵심 기술/솔루션 보유 현황
- 특허, 인증, 자격 보유
- 차별화 기술

**E. 가격 경쟁력**
- 과거 입찰 가격 수준 (가능한 경우)
- 인건비 구조
- 가격 공격적 입찰 성향 여부

**F. 컨소시엄 구성 패턴**
- 주로 어떤 업체와 컨소시엄을 구성하는지
- 파트너 활용 패턴

### 3단계: 경쟁 매트릭스 (정량 비교)

분석 결과를 정량화된 매트릭스로 정리한다:

```
| 평가 항목 | 우리 회사 | 경쟁사 A | 경쟁사 B | 경쟁사 C |
|---|---|---|---|---|
| 유사 실적(건) | 7건 | 12건 | 4건 | 15건 |
| 해당 분야 경력(년) | 25년 | 15년 | 8년 | 20년 |
| 핵심 인력(명) | 5명 | 8명 | 3명 | 6명 |
| 기술 차별화 | LMS/AI | 현지네트워크 | 가격 | 대형실적 |
| 종합 위협도 | — | 높음 | 낮음 | 높음 |
```

### 4단계: 포지셔닝 전략

매트릭스 기반으로 전략을 도출한다:

- **절대 우위 영역**: 모든 경쟁사 대비 우위 → 제안서에서 강조
- **상대 우위 영역**: 일부 경쟁사 대비 우위 → 비교 우위 부각
- **보완 필요 영역**: 경쟁사 대비 열위 → 컨소시엄 파트너로 보완
- **차별화 포인트**: 회사만의 고유 강점
