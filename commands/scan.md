# /scan

조달공고를 분석하고, 시장 트렌드와 연계하여 기회의 전략적 가치를 평가하는 워크플로우.

## 실행 흐름

### Step 0: 컨텍스트 로딩 (context-loader)

`_context/` 폴더가 있으면 먼저 회사 데이터를 로드하여 적합도 판정에 활용한다.

### Step 1: 공고 분석 (procurement-monitor)

사용자가 제공한 공고 정보를 분석한다.

- 공고 텍스트가 첨부된 경우: 핵심 정보 추출 + 요약 카드 생성
- 검색 조건이 제공된 경우: 조건에 맞는 공고를 찾기 위한 가이드 제공
- 복수 공고인 경우: 각각 요약 카드 생성 + 우선순위 매트릭스

**산출물:**
- 공고별 요약 카드 (적합성 신호등 평가 포함)
- 복수 공고 시 우선순위 매트릭스

### Step 2: 시장 맥락 분석 (market-landscape)

공고를 시장의 큰 그림 속에서 해석한다.

- 해당 분야의 시장 트렌드와 연결
- 후속 사업 가능성 평가
- 포지셔닝 기회 식별

**산출물:**
- 해당 공고의 전략적 의미
- 후속 기회 전망
- 권고: 입찰 참여 여부 및 준비 액션
